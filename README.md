# Machine Learning Streamlit App

This project demonstrates Machine Learning concepts using Streamlit and Scikit-learn.

## Features

Regression
- Linear Regression
- R² Score
- Mean Squared Error
- Data Visualization

Classification
- Logistic Regression
- K-Nearest Neighbors
- Accuracy Score
- Confusion Matrix
- Classification Report
- Cross Validation

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

## Run the App

Install dependencies

pip install -r requirements.txt

Run Streamlit

streamlit run streamlit_app.py
